# 将棋画像KIF変換UIの使い方

このUIは、将棋クエストまたはぴよ将棋の一文字駒スクリーンショットを読み取り、局面を確認しながらKIF/SFENをコピーできる確認用画面です。

## 現在の対応範囲

- 対応対象は「将棋クエスト」と「ぴよ将棋」の1枚駒、つまり一文字駒の画面です。
- 対応している入力は、スマホでスクリーンショットを撮影した画像のみです。
- 確認済みの機種は Pixel 7a のみです。
- 他の端末、画面サイズ、テーマ、別アプリ、トリミング済み画像では誤認識する可能性があります。その場合はKIFを他アプリに取り込む前に必ず盤面と持ち駒を目視確認してください。

## いちばん簡単な起動方法

1. 初回だけ、Python 3.10以上を入れます。
2. 初回だけ、このフォルダで次を実行します。

```powershell
pip install -e .
```

3. `start_kif_ui.cmd` をダブルクリックします。
4. ブラウザで `http://127.0.0.1:8765/` が開きます。

将棋クエスト一文字駒向けの学習済みモデルは `models\shogi_quest_ichimonji_piece_model.pkl`、ぴよ将棋一文字駒向けの学習済みモデルは `models\piyo_ichimonji_piece_model.pkl` として同梱しています。`start_kif_ui.cmd` は認識スタイルに合わせてこの2つを自動で使い分けます。

純粋なHTMLファイルだけでは、PC内のPython認識処理を直接実行できません。そのため、このリポジトリでは `start_kif_ui.cmd` がローカルサーバーを起動し、ブラウザ上のHTML UIを開く方式にしています。

## 画像の読み込ませ方

- `画像を選択` からスクリーンショットを選びます。
- PCでスクリーンショットを撮ってクリップボードに入っている場合は、画面上で `Ctrl+V` を押します。
- ブラウザが許可している場合は、`Ctrl+V / 貼付` ボタンからも貼り付けできます。

`認識スタイル` は既定で `将棋クエスト 一文字` です。ぴよ将棋の画像では `ぴよ将棋 一文字` を選んでください。

## 止め方

1. ブラウザのタブを閉じます。
2. `start_kif_ui.cmd` で開いた黒い画面をクリックします。
3. `Ctrl+C` を押します。
4. `バッチ ジョブを終了しますか (Y/N)?` と表示されたら、`Y` を押してEnterを押します。

ブラウザを閉じるだけでは、画像解析用のローカルサーバーは止まりません。分からなくなった場合は、黒い画面の右上の `×` を押して閉じても大丈夫です。

## 確認する場所

- 左側に元画像が表示されます。盤面や持ち駒をクリックすると、対応する画像上の位置が強調表示されます。
- 中央に認識結果の盤面が表示されます。上側に後手の持ち駒、下側に先手の持ち駒を表示します。
- 右側で、選択したマスや持ち駒の認識内容を確認できます。
- KIF欄の `コピー` ボタンを押すと、ボタン表示が `コピー済み` に変わります。

## KIFを他アプリへ渡すとき

1枚のスクリーンショットから出力するKIFは、指し手履歴ではなく局面KIFです。他アプリへ取り込む前に、次を確認してください。

- 盤上の駒配置が元画像と一致していること
- 先手の持ち駒と後手の持ち駒が一致していること
- 手番が正しいこと
- `needs_review` が出ていないこと

取り込み先アプリによっては局面KIFの受け入れ条件が異なるため、KIFが通らない場合はSFENも試してください。
